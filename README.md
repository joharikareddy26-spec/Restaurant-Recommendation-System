# Restaurant Recommendation System

## Objective
Build a content-based recommendation system that suggests restaurants based on cuisine preferences and price range.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Cosine Similarity

## Features
- Restaurant similarity recommendations
- User preference recommendations
- Cuisine filtering
- Price range filtering

## Sample Output

Input:
recommend_restaurant("Le Petit Souffle")

Output:
['Tokyo Mon Amour',
 'Izakaya Kikufuji',
 'Sushi Loko',
 'New Koto',
 'Sushi Leblon']
